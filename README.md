Este programa implementa un Sistema de Gestión de Vehículos, su estructura modela la realidad mediante herencia (creando clases hijas Automovil y Motocicleta que derivan de una clase padre Vehículo) y composición (asociando un objeto de la clase Motor a cada vehículo). El código demuestra el uso  de encapsulamiento a través de decoradores (@property) para proteger los atributos.
